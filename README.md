# IoT Модуль управления освещением с радарным датчиком

![3D модель устройства](https://github.com/VGCH/radar_switch/blob/main/img/lable_text.png)

Работает как самостоятельное утройство, так и в составе уного дома Home Assistant:

В модуле реализованы следующие сервисы

> mDNS

> SSDP/UPnP

> Captive Portal

> MQTT Discovery
